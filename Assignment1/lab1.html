Lab 1

Lab 1 is the first CIS 2336 assignment that we have to submit. It demonstrates the understanding and ability to write basic pages in HTML.

Nowadays, most websites are developed with some PHP/JS framework, like Laravel or JQuery, because writing HTML files from scratch kinda sucks. But hey, you gotta start somewhere!
However, there's still some really important websites that still utilize basic HTML/CSS usage, such as the PNG library.

Pure HTML/CSS websites use to be more popular back then, especially when Geocities was still around. There's a really niche aesthetic that these old websites have that I find interesting.
As time goes on, HTML's importance is superseded by all of the different web libraries that generate the markup for you.
