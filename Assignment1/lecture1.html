Lecture 1

Lecture 1 talked about the history and the inter-operations of the World Wide Web.

There is a lot that can be discussed regarding the journey of data transmission, but to boil it down into a fundamental framework, we can consider the OSI Model.
The Open Systems Interconnect model describes the flow of data from a high-to-low level perspective. There are 7 layers to the OSI model, each of which serve their own set of functionalities.


Lecture 1 also talked about software stacks such as LAMP: Linux Apache MySQL PHP & WISA: Windows IIS SQL ASP.NET
