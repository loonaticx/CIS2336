Lecture 2

Lecture 2 introduced the basics of HTML.

HTML, also known as Hypertext Markup Language, is one of the main components for website functionality.
HTML is not a programming language. It contains no programming logic and only defines the structure of a web page.
The markup language also allows us to insert images, links, lists, headings, metadata, and so much more into a document page. Additionally, it has support for nesting elements.
