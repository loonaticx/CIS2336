Lecture 3

Lecture 3 talked about Cascading Style Sheets, also known as CSS.

CSS is used to make websites not look visually crappy like this one currently does.
The three cascade principles that CSS follows include inheritance, specificity, and location.
Understanding how attribute inheritance works is very important in web design, as it dictates which elements are affected by the CSS rule.
